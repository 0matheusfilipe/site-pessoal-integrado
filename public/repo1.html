Descrição

Nesse trabalho, montamos uma página de um curriculum vitae (CV) aplicando os elementos da linguagem HTML. Incluímos uma foto uma imagem de um avatar que possa representar alguém. No meu caso, compus um currículo fictício. Além disso, inseri dados fictícios para composição dos textos baseados na personagem escolhida. Dessa forma, foi possível fazer um teste completo tanto dos recursos HTML quanto da dinâmica da Web.
