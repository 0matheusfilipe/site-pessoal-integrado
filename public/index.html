Perfil

Aos 20 anos, sou bolsista integral no curso bacharelado em Sistemas de Informação pela Pontifícia Universidade Católica de Minas Gerais (PUC Minas). Possuo experiência em desenvolvimento de software, especialmente em .NET utilizando C#. Além disso, tenho competências técnicas em desenvolvimento web, usando HTML, CSS, JavaScript e Bootstrap 5, como também conhecimentos em banco de dados utilizando SQL Server. Nesse contexto, estou comprometido em aprimorar continuamente minhas habilidades para ingressar no mercado de desenvolvimento .NET e contribuir efetivamente para projetos inovadores.

Destaco-me também em habilidades comportamentais essenciais, como proatividade, trabalho em equipe, comunicação clara e eficaz, pensamento crítico, facilidade com relacionamentos interpessoais e organização. Atualmente, exerço o cargo de estagiário de suporte na empresa Group Software, atuando na linha de frente de análises e solução de dúvidas de clientes na utilização do software para a unidade de negócios Shopping Centers.
